Welcome Marker Whiteboard!

This is a clean canvas designed to work with the Draw and Mark a webpage browser extension.

To start drawing:

1. Click the Web Marker extension icon in your browser toolbar or download it here Draw and Mark a webpage

2. Use the floating toolbar to select your drawing tools

3. Draw, highlight, add text, and annotate freely!

Keyboard Shortcuts:

Shift + D (Pen) • Shift + H (Highlighter) • Shift + E (Eraser) • Shift + T (Text)
